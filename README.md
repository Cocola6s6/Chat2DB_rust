# 启动

* sh ./cli/chat2db.sh start

# 介绍
参考 [Chat2DB](https://github.com/chat2db/Chat2DB) 的效果，使用 chatgpt 进行自然语言翻译，然后对数据库进行操作，使用 rust 语言实现的 web 应用。
详细设计文档请参考：[chat2db_rust 设计文档](rfcs/chat2db.md)
